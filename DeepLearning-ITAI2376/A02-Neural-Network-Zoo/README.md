## Problem Statement

How do you explain complex neural network architectures to someone with no technical background? This assignment challenged students to pick one neural network architecture from the "Neural Network Zoo" and present it creatively — in a way an 11-year-old could understand.

## Approach & Methodology

Each architecture in the zoo was mapped to an animal whose traits mirror how the model behaves:

| Animal | Architecture | Why it fits |
|--------|-------------|-------------|
| 🐆 Cheetah | CNN | Fast at spotting visual patterns |
| 🦝 Raccoon | RNN | Moves through sequences step by step |
| 🐒 Lemur | LSTM | Uses memory to retain long-term context |
| 🦉 Owl | Transformer | Uses attention to focus on what matters |
| 🦫 Beaver | Autoencoder | Compresses and rebuilds information |
| 🦊 Fox | GAN | Creates new data through competition |

**CNN Cheetah** was chosen because CNNs are fast, targeted visual hunters — spotting edges, textures, and objects in images layer by layer, just like a cheetah locks onto its target before striking.

### CNN Architecture Breakdown
- **Convolution layers** — detect local patterns (edges, textures, shapes)
- **Pooling layers** — reduce spatial size while keeping key features
- **Fully connected layers** — combine learned features to make the final prediction

### Feature Learning Progression
```
Early layers  →  Edges and lines
Middle layers →  Textures and shapes
Late layers   →  Objects and classes
```

## Results & Evaluation

- Successfully communicated CNN architecture using visual metaphors accessible to a general audience
- Demonstrated understanding of how CNNs differ from RNNs, LSTMs, and Transformers
- Presented real-world CNN applications: image classification, object recognition, medical imaging, audio pattern detection

## Learning Outcomes

- Understood the layered structure of neural networks (input → hidden → output)
- Learned why CNNs are the go-to architecture for image-based tasks
- Practiced communicating technical concepts to non-technical audiences
- Gained appreciation for why model selection matters — different architectures excel at different data types
