## Problem Statement

Natural Language Processing is one of the most complex challenges in AI — and the 2016 film *Arrival* presents a fictional version of this challenge at its most extreme: decoding an entirely alien language with zero shared context. This assignment explored how real NLP systems face the same core problems the film dramatizes, just at a smaller scale.

## Approach & Methodology

The presentation performed a side-by-side analysis of NLP challenges in the film versus real-world AI systems:

### Key NLP Challenges Identified

| Challenge | In *Arrival* | In Real NLP |
|-----------|-------------|-------------|
| Zero shared context | No common ground with Heptapods | Cold-start translation between low-resource languages |
| Lexical ambiguity | "Weapon" vs "tool" — same symbol, different intent | Word sense disambiguation |
| Pragmatics & intent | Is it a threat or an offer? | Sentiment analysis, sarcasm detection |
| Multimodal communication | Heptapod A (spoken) vs B (written logograms) | Multimodal AI: CLIP, Whisper, GPT-4V |
| Non-linear structure | Heptapod language reflects circular time perception | Non-sequential document understanding |

### NLP Paradigm Evolution Covered
```
Rule-based NLP  →  Statistical models  →  Deep Learning (Transformers)
```

### Modern Tools Connected to the Film
- **GPT-4** — contextual understanding across long documents
- **CLIP** — connecting visual symbols to language meaning
- **Whisper** — speech-to-text across languages
- **Google Translate** — statistical + neural machine translation

## Results & Evaluation

- Produced a 14-slide professional research presentation
- Connected film narrative to concrete NLP concepts: ambiguity, pragmatics, zero-shot learning, and multimodal integration
- Analyzed the limits of current NLP: reward hacking, hallucination, and the gap between syntax and true understanding
- Proposed future directions: multimodal integration, zero-shot cross-lingual transfer, and explainable AI

## Learning Outcomes

- Understood why language understanding is hard — ambiguity, context, and intent cannot be solved with rules alone
- Learned how transformer-based models like BERT and GPT address sequential context through self-attention
- Recognized the gap between pattern matching and genuine language comprehension
- Appreciated how multimodal AI (text + image + audio) is closing that gap
